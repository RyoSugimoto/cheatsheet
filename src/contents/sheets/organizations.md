## 識別管理

### ICANN

Internet Corporation for Assigned Names and Numbers

- [ICANN](http://www.icann.org/)
- 民間の非営利法人で誰でも自由に参加可能
- 本拠地はカリフォルニア州ロサンゼルス、世界各地にオフィス
- ドメイン名、IPアドレスおよびAS番号、ポート番号およびパラメータ番号の割り振り・割り当て
- DNSルートネームサーバの運用・調整
- これらの技術的業務に関するポリシー策定の調整
- [JPNICの「ICANN」解説ページ](https://www.nic.ad.jp/ja/icann/about/organization.html)

### IANA

Internet Assigned Numbers Authority

- ドメイン名、IPアドレス、プロトコルなど、インターネット資源のグローバルな管理
- 現在「ICANN」がその役割を引き継ぎ、IANAはICANNの機能（下部組織）の名称
- [IANA](https://www.iana.org/)

### 地域レジストリ

地域におけるIPアドレスとAS番号の管理

- [APNIC](https://www.apnic.net/): アジア太平洋圏、オーストラリアに事務所
- [ARIN](https://www.arin.net/): 北米およびカリブ海
- [RIPE NCC](https://www.ripe.net/): ヨーロッパおよび中東
- [LACNIC](https://www.lacnic.net/): 南米
- [AFRINIC](https://afrinic.net/): アフリカ

### JPNIC

[一般社団法人 日本ネットワークインフォメーションセンター](https://www.nic.ad.jp/

- 「APNIC」から割り振られたIPアドレスやAS番号を日本国内で管理
- 研究、調査、啓発イベント、情報発信など
- `.jp` ドメインの登録・管理業務は「JPRS」に移管

### JPRS

[株式会社 日本レジストリサービス](https://jprs.co.jp/)

- TLD `.jp` のレジストリ

## インターネット技術

### ISOC

Internet Society

- インターネット技術およびシステムに関する標準化、教育、ポリシーに関する課題や問題を解決あるいは議論
- 非営利の国際組織
- 組織会員、個人会員、世界で90以上の支部
- 日本支部（ISOC-JP）

### IAB

Internet Architecture Board

- 「ISOC」の技術理事会
- インターネット関連の技術者の集団
- 「IETF」に方向性を示す

### IETF

Internet Engineering Task Force

- [IETF](http://www.ietf.org/)
- インターネット技術の標準化を推進する任意団体
- 技術仕様書「RFC（Requests for Comments）」

## ウェブ技術

### W3C

World Wide Web Consortium

- [W3C](https://www.w3.org/)
- World Wide Web（ハイパーテキスト）に関する技術を標準化する、非営利団体
- ティム・バーナーズ＝リーが創設
- 現在、HTMLとDOMの策定は「WHATWG」が担う

### WHATWG

Web Hypertext Application Technology Working Group

- [WHATWG](https://whatwg.org/)
- HTMLと関連技術（Web APIなど）の開発のためのコミュニティ
- Apple、Mozilla、Opera、Microsoft
- 「HTML Living Standard」を提唱 -> 現在のHTMLの唯一の標準
